## Pattern

1. Start from a **real, elegant mathematical object**.  
2. Notice a **numerical coincidence** or a **geometric fact**.  
3. **Declare** that the object *is* a physical law / control mechanism / frequency of dark energy / optical degree of freedom.  
4. Skip the **derived map** object → measurable (equations, operators, units, amplitude).  
5. Optionally dress the claim in advanced vocabulary (Coxeter groups, holography, time-reversal).

## Filter (use before writing a paper sentence)

- [ ] Is there an equation connecting the object to an observable?  
- [ ] Are units and amplitudes stated?  
- [ ] Does current data **prefer** or only **tolerate** the claim?  
- [ ] If the best fit is null, is the prose still written as a discovery?  

If any box fails, the sentence is **narrative**, not a result.
